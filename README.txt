A4P Diagnostic Hub

Contenu du pack
- index.html : page d'accueil centrale du hub A4P
- synthese.html : page de synthèse structurante
- assets/logo-a4p.jpeg : logo intégré

Utilisation
1. Créer un nouveau dépôt GitHub Pages
2. Déposer tous les fichiers du dossier à la racine du dépôt
3. Activer GitHub Pages
4. Ouvrir l'URL publiée

Limite actuelle
Le hub réunit les trois tests dans un seul environnement visuel, mais la remontée automatique des résultats n'est pas encore branchée. Pour cela, il faudra harmoniser les variables de sortie des trois tests.
